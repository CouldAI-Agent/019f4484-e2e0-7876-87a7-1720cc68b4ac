# Personal Knowledge System

A cross-platform, private, offline-first personal encyclopedia. 

This repository implements the cosmic dashboard and lock screen as requested in the project vision. It is fully functional on iOS, Android, Web, Mac, Windows, and Linux.

## Features
- **App Lock**: Secured with a master key (`Picture@_Lab5^6#`).
- **Cosmic Dashboard**: A stunning start screen with stats, recent pages, and quick actions.
